# প্রশ্ন ১: টাইপস্ক্রিপ্টে keyof কীওয়ার্ডের ব্যবহার কী?

keyof হলো একটা বিশেষ অপারেটর যেটা কোনো টাইপের সব প্রোপার্টি র নাম মিলিয়ে একটা ইউনিয়ন টাইপ তৈরি করে। মানে, keyof সেই টাইপের সব কী-গুলোকে স্ট্রিং লিটারেল ইউনিয়ন হিসেবে রিটার্ন করে।

উদাহরণস্বরূপ, একটা User টাইপ আছে, যেখানে name এবং age প্রোপার্টি  রয়েছে। keyof User লিখলে সেটা "name" | "age" টাইপ তৈরি করবে। এটার প্রধান সুবিধা হলো ফাংশন লেখার সময় শুধু মাত্র সেসব key কে allow করা যায় যারা ভ্যালিড।

যেমনঃ
```typescript
type User = {
  name: string;
  age: number;
};
```

আমরা লিখতে পারিঃ
```typescript
type UserKeys = keyof User;
```

এতে করে এখন UserKeys শুধু মাত্র name আর age -ই হতে পারবে।

# প্রশ্ন ২: ইউনিয়ন এবং ইন্টারসেকশন টাইপের উদাহরণ

টাইপস্ক্রিপ্টে টাইপ কম্বাইন করার - মানে একাধিক টাইপ মিলে একটা বানানোর ২টা প্রধান উপায় আছে - ইউনিয়ন এবং ইন্টারসেকশন।

## ইউনিয়ন টাইপ

ইউনিয়ন টাইপ পাইপ সিম্বল ( | ) দিয়ে লেখা হয়, এবং এটা বোঝায় যে ভ্যালু এই টাইপগুলোর যেকোনো একটা হতে পারে। উদাহরণস্বরূপ, একটা Status টাইপ তৈরি করা যায় যেটা "success" বা "error" বা "loading" হতে পারে। এখন কোনো ভেরিয়েবলের টাইপ Status দিলে সেটা শুধু এই তিনটা মানের মধ্যে একটা নিতে পারবে। অন্য কোনো স্ট্রিং দিলে টাইপস্ক্রিপ্ট এরর দেখাবে।

## ইন্টারসেকশন টাইপ

ইন্টারসেকশন টাইপ অ্যাম্পারস্যান্ড সিম্বল (&) দিয়ে লেখা হয়, এবং এটা একাধিক টাইপের সব প্রোপার্টি  একসাথে কম্বাইন করে। যেমন একটা Person টাইপে name প্রোপার্টি  আছে এবং একটা Employee টাইপে salary এবং employeeId প্রোপার্টি  আছে। এখন Person এবং Employee এর ইন্টারসেকশন করলে নতুন টাইপে তিনটা প্রোপার্টি ই থাকবে - মানে, name, salary এবং employeeId। এই টাইপের কোনো অবজেক্ট তৈরি করতে হলে সব প্রোপার্টি ই দিতে হবে, দেওয়া বাধ্যতামূলক হয়ে যাবে, নাইলে টাইপ এরর হবে।
